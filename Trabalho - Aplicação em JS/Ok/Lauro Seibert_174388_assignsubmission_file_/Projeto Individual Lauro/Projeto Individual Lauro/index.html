Gerador de Escala de Cores HSL

Utilizando o sitema de cores HSL, o programa gera uma escala de cores baseada na entrada do usuário, variando de 0 até 359. O usuário pode copiar as cores no formato RGB para uso próprio. Presets de cores estão disponíveis, para aplicá-los é necessário que o campo Valor HSL esteja vazio, e o botão "Aleatório" gera uma escala aleatória para o usuário.
